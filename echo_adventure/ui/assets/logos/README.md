# ECHO Logo Assets

The game uses the ECHO logo files in this directory.

Expected filenames:

- `echo-logo-full.png` for the full primary logo
- `echo-logo-small.png` for the smaller compact logo or mark

The browser app serves these files as:

- `/ui/assets/logos/echo-logo-full.png`
- `/ui/assets/logos/echo-logo-small.png`
